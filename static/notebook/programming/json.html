Python


• json파일을 만들고 (dump), json파일을 가지고오기 (load)
• string obj를 만들기 (dumps)
• dumps와 loads도 기본적으론 같은데, 파일이 아닌 py obj들을 할 경우
import json
with open(filename, 'w', encoding="utf-8") as f:
json.dump(data, f, ensure_ascii=False, indent="\t")
with open(filename) as f:
data = json.load(f)
# encoding, ensure_ascii, indent 등이 없어도 되긴 함
