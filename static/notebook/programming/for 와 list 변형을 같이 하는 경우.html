for 와 list 변형을 같이 하는 경우

for 내부에서 list를 수정하는 경우 아무 것도 안하고 하면 문제가 생길 수 밖예 없음.

예를들어 students 라는 list 에서 friend, Robert를 삭제하는 다음과 같은 함수의 경우,
첫번째 loop에서 students[0] 에서 Robert=0 을 삭제하고 나면,
student[1] 에 있던 friend=1 이 student[0] 자리로 들어가서 두번 째 loop에서 inspect 자체가 안됨
# students = [0,1,2,3,4,5], friend=1, Robert=0
def removePairFromStudents(students, friend, Robert):
for student in students:
if student == friend:
students.remove(friend)
if student == Robert:
students.remove(Robert)
return students
