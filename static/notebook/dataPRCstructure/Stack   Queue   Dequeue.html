Stack / Queue / Dequeue

• 선형적인 (계층이 없는) 자료를 표현하기 위한 자료구조
    Stack   : in == out
    Queue   : in != out
    Dequeue : stack 이 양쪽에서 벌어진다고 생각하면 될 듯

    

• 구현
    ◦ linked list

        (1) Stack
            

        (2) Queue
           

        (3) Dequeue
            

    ◦ Dynamic Array

        (1) 일반 dynamic array 를 사용할 때의 문제점
            stack 의 경우는 push, pop 을 O(1) 으로 구현 가능한데
            queue 에서 pop 을 하거나
                

            dequeue 의 앞에서 push/pop 을 할 경우 O(n) 이 걸림
            

        (2) 환형버퍼 Circular Buffer
            밀고당기는게 없도록 하기 위해, 아예 head, tail 을 그냥 저장하는 길을 택하게 되는데,
            그렇게 하면 앞부분을 못쓰니까, 앞이 뒤가 되도록 circular 하게 배열을 만듦
