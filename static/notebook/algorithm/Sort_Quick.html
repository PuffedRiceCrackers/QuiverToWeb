Sort_Quick

Quick Sort 자체는 다음의 심플한 방법을 따르는 Divide and Conquer 알고리즘임

    1. Pivot을 선택하고, pivot의 left=smaller, right=bigger이 되도록 어떻게든 놓는다
    2. left subgroup 을 quicksort 한다
    3. right subgroup 을 quicksort 한다

        

        

그러나 유튭보면 알겠지만, 1을 구현하는 방법이 다 달라서 헷갈리는거임

(1) iterator와 swap하는 방법

    • ls : pivot보다 작은 그룹중 last element의 idx
         : 논리적으로 볼 때, ls=-1로 시작해야 맞음 (ls=0으로 놓으면, a[0]이 pivot보다 큰 경우 안되니까)
         : 마지막에 ls+1 에 pivot이 들어감 (ls의 정의가 pivot보다 작은 애의 마지막 idx니까) 

        ls 는 작을 때만 ++
        i 는 당연히 매번 ++

        - ⚽️ 큰 경우 : i++
        - 🏀 작은 경우 : ls++, swap(i,ls), i++

(1) 번 코드
def partition(A, start, end):
wall = start - 1
pivot = A[end]
for i in range(start, end):
if A[i] <= pivot:
wall += 1
A[wall], A[i] = A[i], A[wall]
A[wall + 1], A[end] = A[end], A[wall + 1]
return wall + 1
def quickSort(A, start, end):
if start < end:
wall = partition(A, start, end)
quickSort(A, start, wall - 1)
quickSort(A, wall + 1, start)
return A
print(quickSort([72, 30, 5, 9, 34], 0, 4))
