While 신중히 쓰기

① While에 탈출조건을 여러개 달 때 신중해야 함
    ◦ 예를 들어 while 탈출조건이 2개만 되어도 고려하게 되는 가지수가 4가지 임
     
    ◦ 저 중에 a만 while안에 포함하게 될 건데, 그럼 나머지 3개에 대한 처리를 밖에서 해줘야 된다는 말
      경우에 따라서 쉽게 될 수도 있지만 그게 아니라면 벤다이어그램이라도 진짜 그려보는게 좋음
while min(scoville) < K and len(scoville) >= 2: # a
...
if min(scoville) >= K: # y,z
return count
else: # x
return -1
    ◦ while loop 에 조건을 써버리면 and or밖에 안되지만, while 내부에 쓰면 elif 도 쓸 수 있고
      fine tuning이 가능하니 때에 따라서는 이게 더 좋을 듯함
