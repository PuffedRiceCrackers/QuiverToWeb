enum

• type 의 일부로, 정해진 case만을 가짐 (때로는 이에 딸려있는 associated data도)
• which case , what associated date : 이거 하나가 state를 이룸 
• value type
• store되는 value는 state까지이고, var 등의 값은 computed value 임 (stored value는 가질 수 없다?) 
여기까진 다른 lang랑 똑같은데, swift 에서 enum은 associated value를 가질 수 있음
enum fastFoodMenuItem{
case hamburger(numOfPatties:Int)
case fries(size: FryOrderSize)
case drink(String, ounces:Int)
case cookies
}
enum fryOrderSize{
case large
case small
}
/**/
let menuItem:fastFoodMenuItem = fastFoodMenuItem.hamburger(patties:2)
let anotherItem:fastFoodMenuItem = fastFoodMenuItem.cookies
let finalItem:fastFoodMenuItem = .cookies
Check state of enum : Switch
let menuItem:fastFoodMenuItem = fastFoodMenuItem.fries(size: .large)
switch menuItem{
case .hamburger :
break // do nothing
case .cookies :
print("cookies") // fastFoodMenuItem 생략가능
default: print("other")
}
Access associated data of enum : Switch + Let
let menuItem:fastFoodMenuItem = fastFoodMenuItem.fries(size:.large)
switch menuItem{
/* enum 할 때의 이름과 같을 필요 없음 */
case .hamburger(let patty):print("patty : \(patty)")
case .friest(let size):print("size : \(size)")
case .drink(let brand, let ounces):print("brand : \(brand), ounces:\(ounces)")
case .cookies:print("cookies")
}
