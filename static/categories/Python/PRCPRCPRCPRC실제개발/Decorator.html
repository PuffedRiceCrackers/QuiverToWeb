Decorator
    • function 또는 class 의 기능을 바꿔주거나 추가
    • 사실상 일은 inner가 함
    • 만드는 방법
        ① wrapper(func)
        ② 내부에 inner()
        ③ inner()에서 func를 실행한다
        ④ inner와 func의 인자를 맞춘다
        ⑤ innter를 return한다
def wrapper(func):
def innter(x):
func(x)
print("-------")
return inner
wrapper(func)('hello') # 라고도 쓸 수 있지만 좀 구리니까
@wrapper
def func(x):
print(x) # func(x) 하면 자동으로 wrapper가 적용된 걸로 나옴
