• html, css, javascript 연결
    ◦ flask 가 찾을 수 있도록 url_for 을 쓸 것 (html만으로는 path를 제대로 써도 못 찾은 적이 있음)
    ◦ html 에 favicon.ico 와 style.css를 포함시키지 않으면
      request의 path로 favicon.ico, style.css가 날라오게 되므로, favicon과 css가 없더라도 처리해줄 것
    ◦ html 의 head 에 css path를 추가하고, style에 javascript path 추가
<link rel="stylesheet" type="text/css" href= "{{ url_for('static',filename='css/style.css') }}">
<link rel="shortcut icon" type="image/x-icon" href="{{ url_for('static', filename='favicon.ico') }}">
<script src="./static/script.js"></script>
<script src="{{ url_for('static', filename='script.js')}}"></script>
