rauth 모듈이 안됨

일단 내 api에서 허용된 redirect_url 은 이렇게 다양했음
• http://127.0.0.1:5000/loginAuthorized/ ✨
• http://127.0.0.1:5000/loginAuthorized
• http://127.0.0.1:5000//loginAuthorized/
• http://127.0.0.1:5000//loginAuthorized
실제로 연결되는 건 첫번째 url 이랑, 다음 url 임
    ◦ 이유는 모르겠지만 //loginAuthorized가 됨 (%2F%2FloginAuthorized)
    ◦ loginAuthorized 끝에 /가 그대로 살아있는 채로 주어야 함
    ◦ 그래도 http://127.0.0.1:5000/loginAuthorized/ 에 연결은 됨
https://kauth.kakao.com/oauth/authorize?
client_id=79e7dabc64dc86841b6ecb88dab11244&
redirect_uri=http%3A%2F%2F127.0.0.1%3A5000%2F%2FloginAuthorized/&
response_type=code
이걸 rauth 를 써서 만들면
from rauth import OAuth2Service
@app.route('/')
def main():
service = OAuth2Service(
client_id='CLIENT_ID',
client_secret='',
name='kakao',
authorize_url='https://kauth.kakao.com/oauth/authorize'
)
params = {
"client_id": "CLIENT_ID",
"redirect_url": "http://127.0.0.1:5000/loginAuthorized/",
"response_type": "code"
}
url = service.get_authorize_url(**params)
return render_template('index.html', request_url=url)
url 이 다음과 같이 만들어짐
    ◦ loginAuthorized 끝이 /가 아니라 %2F
https://kauth.kakao.com/oauth/authorize?
client_id=79e7dabc64dc86841b6ecb88dab11244&
redirect_url=http%3A%2F%2F127.0.0.1%3A5000%2F%2FloginAuthorized%2F&
response_type=code
그래서 아 뒤에 /여야 되는데 %2F 로 바뀌어서 안되나보다 싶어서 
http://127.0.0.1:5000/loginAuthorized
http://127.0.0.1:5000//loginAuthorized
이런 url 을 추가한 다음에 redirect_url 에 /없이 해서 만들어봤는데도 안됨. 걍 안 됨
