sort, sorted

iterable.sort(*, key=None, reverse=False)
    : change the original, no return 
sorted(iterable, *, key=None, reverse=False)           
    : deesn’t change the original, return

둘다 key를 사용하여, 묶여있는 데이터중 하나를 기준으로 소트할 수 있으므로 기억하도록 하자
a = [1,10,3]
a.sort(reverse=True)
>> a # [10,3,1] 원래 data가 변경됨
참고로 dictionary는 a.sort()가 안됨. 그래서 sorted 를 해줘야 되는데..
a = {"a":1, "b":100, "c":50}
a1 = sorted(a.items(), key=lambda x:x[1], reverse=True) # dict의 value로 sort
a2 = sorted(a.items(), key=lambda x:x[0], reverse=True) # dict의 key로 sort
# key(func)가 iterable을 arg로 받고, 거기서 나온 return값을 기준으로 소팅해줌
주의점
    만들면서 소팅하는 건 안되고, slicing 하면서 소팅하는 것도 안되고,
    완전한 리스트를 소팅해야 함
# depthList 가 None이 뜸
depthList = ([10, 4, 2]).sort(reverse=True)
# depthList 가 제대로 나옴
depthList = [10, 4, 2]
depthList.sort(reverse=True)
temp = array[i - 1:j].sort() # temp = None
