max, min (응용)

max(iterable, *key)
    • max, min 은 아무 조건이 없다면 다차원에 대해서도 가장 낮은 차원만으로 min, max를 따짐
        (예) [[5,’고군’],[28,[‘광은’]] 에 대해서 max 하면, 5와 28만 비교
    • iterable 중 정확히 어떤 것을 비교하여 max, min을 구할 것이냐 지정해줄 때 key 라는 arg를 사용
max(a, key=lambda x:f(x))
        - a (iterable) 의 각 element가 lambda의 x로 들어감
        - lambda의 return 값들을 모두 계산하여
        - return 값의 max에 해당하는 원래 iterable의 element를 반환함
• 이걸 알아야 싶기도 한게, 그냥 list comprehension 해서 새로 만드는 게 더 빠르긴 함
    * key arg 활용
    

    * list comprehension
    
• dictinary 에 응용해보자
safeDishes = {'cl': 3, 'bom': 2, 'dara': 3, 'minzy': 3}
max(safeDishes.items(), key=lambda x:x[1]) # ('bom', 2)
