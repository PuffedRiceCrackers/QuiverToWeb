stack

        ◦ 뒤에 넣기   : .append()
        ◦ 뒤에서 빼기  : .pop()
a = [1, 2, 3]
a.append(4) // [1,2,3,4]
a.pop() // [1,2,3]
Queue, Deque
    뭐가 되었든 python의 deque를 사용하도록 한다
        ◦ from queue import Queue는 애초에 thread message exchange를 위해서 만들어져있고 내부는 deque˚
        ◦ 그저 data structure를 원하는 것이라면 deque를 쓰는게 훨씬 빠르고 (※ 참조)
        ◦ 문법도 list랑 비슷해서 익숙함
    ※ 속도비교 (100번 넣고 100번 pop함)
        * Queue >> 457 µs ± 3.09 µs per loop (mean ± std. dev. of 7 runs, 1000 loops each)
        * deque >> 18.9 µs ± 3.29 µs per loop (mean ± std. dev. of 7 runs, 10000 loops each)
    구현
        ◦ 앞에 넣기 : .appendleft()
        ◦ 뒤에 넣기 : .append()
        ◦ 앞에 빼기 : .popleft()
        ◦ 뒤에 빼기 : .pop()
from collections import deque
# que
a = deque([1, 2, 3])
a.appendleft(0) # [0,1,2,3]
a.popleft() # [1,2,3]
a.append(4) # [1,2,3,4]
a.pop() # [1,2,3]
    ※ 참고로 deque로 slicing을 하고자 할 때는 일반적인 방법으론 안 통하니 itertools.islice() 를 쓰도록
# ❌ 에러남
a= deque([1, 2, 3, 4, 5]
for i in a[1:3]:
print(i)

print(i)
