입출력

입력
    split() : 띄어쓰기/separator가 있는 걸 구분해서 받고 싶을 때 (기본적으로 list 반환)
    map     : 한 개 이상의 input을 int등으로 받고 싶을 때

기본입력 (str으로 받아짐)
myStr = input()
>> 1
>> myStr = '1'
띄어쓰기된 한줄의 입력을 나눠받아 str list로 저장 .split()
myName = input().split() # 기본적으로 list로 저장해줌
>> Trevor Noah
>> myName = ['Trevor','Noah']
띄어쓰기된 한줄의 입력을 나눠받아 int list로 저장
intList = list(map(int, input().split())) # list()안해주면 map object됨
>>> 1 3 4
>>> intList = [1,3,4]
띄어쓰기된 한줄의 입력을 나눠받아 multiple var에 저장
# str으로 저장
first, second, third = input().split()
>> 1 2 3
>> first = '1', second = '2', third = '3'
# int로 저장 # list casting을 안해도 되는게 신기하구만
a, b, c = map(int, input().split()) # int(input().split())이 안됨
>> 1 2 3
>> a=1, b=2, c=3
띄어쓰기 없는 한줄을 입력받아 multiple var에 저장
a,b,c = list(input()) # a,b,c=input().split()은 여기서 불가능. 구분자 없는 input이므로
>> 123
>>> a='1', b='2', c='3'
※ map 에 대한 이해
    map(function, iterable)
    iterable한 data set을 function에 넣은 return값으로 map object를 만들어 줌
    (여러 var로 받으면 해당 type으로 주기도 함)
def plusHundred(a):
return a + 100
>>> list(map(plusHundred, [1,2,3]))) = [101, 102, 103]
출력
① 포맷팅 출력

b = 2
print(a,b) # 1 2
③ 이어서 출력
print('a',end='')
print('b') # ab 로 출력됨
