Math 수학

• infinite
from math import inf
print(inf+3) >> inf
• log
    가장 광범위하게 쓸 수 있는 건 log(x,base) 이지만, 이상한 floating error가 있음
from math import log
>> log(1000,10) # 2.9999
    log2, log10 모듈이 더 정확함
from math import log2, log10
>> log2(8) # 3
>> log10(1000) # 3
• 진수
    ◦ int(x, base=y)
        • y진수로 나타내진 숫자 x를 넘겨주면, 10진수 int 형으로 변환
            ◦ x가 number, base X : float 등을 int로 바꿔줌
            ◦ x가 str, base O    : 10진수 형태로 바꿔준다 (number 타입을 넘기면 안됨) 
int(3.8) >> 3
int(3.8, base=10) >> error
int('1111',2) >> 15
    ◦ 10 진수 → 2 진수
a = "{0:b}".format(15) >> '1111'
• 연산
    ◦ 나누기 /, //
        /  : 결과는 float
        // : 결과가 int (나머지는 버림)
6/2 >> 3.0
6//2 >> 3
5//2 >> 2
    ◦ 나머지 %
6%3 >> 0
5%3 >> 1
2%3 >> 2 # 작은 수를 큰 수로 나누면, 작은 수로 나옴
    ◦ 제곱 (**) 과 루트 (sqrt) 
① **
print(3**2)
print(3**0.5)
%%timeit 해보면 그렇게 빠른 연산은 아닌 것 같음
② 제곱, 제곱근
from math import sqrt
a = pow(2, 3) # log_2(3)
b = sqrt(9) # 제곱근
    ◦ 반올림 
>> round(3.65) # 4
>> round(3.2) # 3
