Class

• 파이썬의 모든 class 는 Object 를 기본적으로 상속
• 파이썬에서 class 는 추상적인 어떤 schema 가 아니라, 실제 메모리로 접근가능한 object 이다 (그 자체가 객체임)
hex(id(cat.__class__)) # '0x7fd5b47eada8'
• class 변수는 class 내부, class method 내에서 정의되고
  instance 변수 instance method 안에서 정의된다.
• class가 instance variable을 바로 접근하지 못하는 것을 제외하고
  (이건 당연함. instance 개수만큼 instance variable이 생길테니), 서로 다 접근가능하다.
• method 의 첫번째 arg 는 관례상 self 가 오는 것이지, 사실상 아무거나 써도 된다
