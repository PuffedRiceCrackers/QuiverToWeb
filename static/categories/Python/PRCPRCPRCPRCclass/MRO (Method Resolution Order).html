MRO (Method Resolution Order)

http://python-history.blogspot.com/2010/06/method-resolution-order.html

• 기본적인 python class 의 (MRO : Method Resolution Order) 는 left-to-right의 dfs임

    

이런식으로 inheritance가 있으면, D에 없는 method를 찾을 때 B → A → C 이런 식으로 찾게 됨 

• Diamond 인 경우는 중복 되는 걸 제외해주고 찾게 됨 (B→A→C→A 가 아니라, 한번 나온 A는 제외해서 B→A→C)

    

• class.mro() 또는 class.__mro__
    method resolution order를 프린트해줌. 위와 같이 다이아몬드 상황을 만들었을 때.
class A:
pass
class B(A):
pass
class C(A):
pass
class D(B,C):
pass
D.mro()
>> [__main__.D, __main__.B, __main__.C, __main__.A, object]
D.__mro__
>>(__main__.D, __main__.B, __main__.C, __main__.A, object)
